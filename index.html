Ping-Pong - Player VS Computer

Este jogo de Ping Pong foi desenvolvido com p5.js, uma biblioteca JavaScript voltada para criação gráfica e interações visuais. O objetivo foi recriar a experiência clássica de Pong, onde o jogador controla uma raquete e enfrenta uma IA simples

A lógica do jogo inclui movimentação do jogador, controle da bola com rebote nos limites da tela e uma inteligência artificial básica que segue a bola. Tudo isso com física simples, controle fluido e uma pitada de nostalgia arcade.

Flappy Bird

Este jogo foi desenvolvido no Scratch, uma plataforma de programação visual voltada para iniciantes. Utilizando blocos de código arrastáveis, o projeto recria a mecânica clássica de Flappy Bird — clique para fazer o personagem voar e desvie dos canos!

Foram usados conceitos como detecção de colisão, movimentação com gravidade simulada, clonagem de obstáculos e pontuação automática. Tudo isso sem escrever uma linha de código, provando que é possível criar jogos divertidos de forma simples e intuitiva!

Vale dos Baixinhos

O site Vale dos Baixinhos foi desenvolvido com as linguagens base da web: HTML para estruturar o conteúdo e CSS para estilizar cada detalhe com charme e humor. Cada seção foi pensada para destacar os “produtos” únicos da loja com um visual leve, responsivo e divertido.
